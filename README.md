# DualStream TransUNet Implementation

## 📌 Overview

This repository contains the implementation of a DualStream TransUNet model for image segmentation tasks. The project is based on advanced deep learning techniques combining Convolutional Neural Networks (CNNs) and Transformer-based architectures to improve segmentation accuracy.

## 🎯 Objective

The goal of this project is to enhance feature extraction and contextual understanding by using a dual-stream architecture integrated with TransUNet, enabling better performance on complex image datasets.

## 🧠 Methodology

* Dual-stream architecture for parallel feature extraction
* CNN backbone for capturing local features
* Transformer encoder for global context understanding
* Decoder for precise segmentation output

## 📊 Results

* Improved segmentation accuracy compared to baseline models
* Better performance on complex and noisy datasets
